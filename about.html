Acerca de MS-39 Viewer

¿Qué es MS-39 Viewer?

MS-39 Viewer es una plataforma web estática diseñada para visualizar estudios corneales de manera profesional, simple e interactiva. La plataforma permite explorar visualizaciones 3D generadas con Plotly en un entorno limpio y accesible.

Este proyecto proporciona una interfaz moderna para navegar entre múltiples estudios, con funcionalidades de búsqueda, filtrado y visualización optimizada para dispositivos de escritorio y móviles.

Características Principales

  • Visualizaciones Interactivas: Gráficos 3D generados con Plotly que permiten rotar, hacer zoom y explorar los datos.
  • Búsqueda y Filtros: Sistema de filtrado por tipo de gráfico, clase y número de fila para encontrar estudios rápidamente.
  • Diseño Responsivo: Interfaz optimizada para todos los tamaños de pantalla, desde móviles hasta escritorios de alta resolución.
  • Modo Oscuro: Cambio automático o manual entre temas claro y oscuro, con persistencia de preferencias.
  • Accesibilidad: Cumple con estándares WCAG 2.1 AA, incluyendo navegación por teclado, contraste adecuado y compatibilidad con lectores de pantalla.
  • Compartir Enlaces: Copie enlaces directos a visualizaciones específicas para compartir con colegas.

Tecnología

MS-39 Viewer está construido como un sitio web estático utilizando tecnologías web estándar:

  • HTML5: Estructura semántica y accesible.
  • CSS3: Estilos personalizados sin frameworks pesados, con variables CSS y diseño responsivo.
  • JavaScript Vanilla: Lógica de aplicación sin dependencias externas.
  • Plotly.js: Biblioteca de visualización para gráficos interactivos (incluida en los archivos HTML individuales).

El sitio es completamente estático y puede alojarse en cualquier servidor web, incluyendo GitHub Pages, Netlify, Vercel o AWS S3.

Privacidad y Seguridad

Protección de Datos

MS-39 Viewer ha sido diseñado con la privacidad como prioridad. Todas las visualizaciones son completamente anónimas:

  • No se incluye ni se expone información de identificación de pacientes (patient_ID u otros datos sensibles).
  • Los datos mostrados en las visualizaciones están agregados y anonimizados.
  • No se recopilan datos de usuarios ni se utilizan cookies de seguimiento.
  • El sitio no se conecta a servidores externos excepto para cargar los gráficos locales.

Almacenamiento Local

El sitio utiliza localStorage únicamente para guardar la preferencia del tema (claro/oscuro). Esta información permanece en su dispositivo y nunca se transmite a ningún servidor.

Seguridad

Todas las visualizaciones están embebidas mediante iframes con políticas de referrer restrictivas (no-referrer) para proteger la privacidad de navegación. El sitio puede servirse de forma segura mediante HTTPS.

Cómo Usar

  1. Explorar Estudios: Visite la página de Estudios para ver todos los gráficos disponibles.
  2. Filtrar: Use los controles de búsqueda y filtros para encontrar estudios específicos por clase, tipo o fila.
  3. Visualizar: Haga clic en "Ver Gráfico" en cualquier tarjeta para abrir el visor interactivo.
  4. Interactuar: En el visor, puede rotar, hacer zoom y explorar el gráfico 3D usando el mouse o pantalla táctil.
  5. Compartir: Use el botón "Copiar Enlace" para compartir una visualización específica.
